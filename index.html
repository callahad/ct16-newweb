Resources:

  • Service Worker Cookbook, with many annotated demos of using service works in various ways, including push notifications, offline support, and more.
  • Offline Cookbook, which comprehensively describes strategies for handling interactions between the network, service workers, and caches.
  • Instant Loading: Building offline-first Progressive Web Apps - Google I/O 2016, an exceptional talk by Jake Archibald that incrementally converts a traditional web app to one that makes full use of service workers and manifests.
  • MDN: Using Service Workers
  • MDN: Service Worker APIs
  • MDN: Push API
  • Mozilla Hacks: Web Push Arrives in Firefox 44
  • Bug 1285858, support for display: standalone in Firefox for Android

Experiments and Examples

  • Pokedex.org, Nolan Lawson's offline, PWA-style Pokedex
  • Ambient badging and “pop into browser”, Opera's experiments with the transitions between web-like and app-like experiences.
  • sw-delta, Gaël Métais's experiment using service workers to patch cached files instead of retransmitting the whole thing.
  • URLs into IndexedDB, Joshua Bell's experiment using service workers to fetch data from IndexedDB based on URLs.
  • PWA.Rocks, an index of websites using Service Workers and App Manifests.
